Jawab

1. Gunakan Rumus kekekalan momentum
  1. Langkah 1 - Tulis Rumusnya
    M1 . V1 + M2 . V2 + M3 . V3 + M2 . V4
  2. Langkah 2 - Masukan angkanya
    {{ initial[0].mass }} . {{ initial[0].vX }} + {{ initial[1].mass }} . {{ initial[1].vY }} + {{ initial[2].mass }} . {{ velocity(initial[2].vY, initial[2].vX) }} + {{ initial[3].mass }} . {{ velocity(initial[3].vY, initial[3].vX) }}
  3. Langkah 3 - Hitung
    Momentum totalnya adalah: {{ (initial[0].mass * velocity(initial[0].vY, initial[0].vX)) + (initial[1].mass * velocity(initial[1].vY, initial[1].vX)) + (initial[2].mass * velocity(initial[2].vY, initial[2].vX)) + (initial[3].mass * velocity(initial[3].vY, initial[3].vX)) }} kg m/s

Note:

Momentum total sebelum dan sesudah tumbukan hasilnya akan tetap, karena gaya gesek ditiadakan. Jika gaya gesek diadakan maka bola akan diperlambat secara linier.
